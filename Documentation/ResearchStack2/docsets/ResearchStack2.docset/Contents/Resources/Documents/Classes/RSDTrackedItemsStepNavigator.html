RSDTrackedItemsStepNavigator

open class RSDTrackedItemsStepNavigator : Decodable, RSDStepNavigator

RSDTrackedItemsStepNavigator is a general-purpose navigator designed to be used for selecting tracked data such as medication, triggers, or symptoms.
